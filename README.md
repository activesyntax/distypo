DisTypo is a typography corrector. You paste in your text, and the application points out the small mechanical mistakes that spell-checkers ignore — the wrong dash, the missing space, the straight quote in a paragraph of curly ones. It does not rewrite your prose. It only fixes what is unambiguously wrong.
